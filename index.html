Notation Guide

section: length key tempo - e.g., a1: 36 b3 120

- track: instrument volume pan - e.g., - vocal: lead_6_voice .5 .5

section.track: offset - e.g., a1.vocal: 0 or a1.vocal: -2

- | rhythm | - Note durations, e.g., - | .5 .5 1 |

- | melody | - Notes (c4, d4), rests (.), ties (-), chords (c4-e4-g4)

- | lyrics | - Optional text for each note

Percussion: Use xb (bass drum), xs (snare), xhc (closed hi-hat), xh (open hi-hat), xc1 (crash), etc.

Instruments: acoustic_grand_piano, electric_guitar_clean, acoustic_bass, violin, string_ensemble_1, choir_aahs, trumpet, flute, lead_6_voice, percussion, and more!
